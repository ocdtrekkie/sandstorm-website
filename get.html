Details

All about grains

A grain is a document, chat room, mail box, notebook, blog, or anything else you create.

Resource limits

Sandstorm apps only use RAM while you are using them. Sandstorm uses compute units as a measurement of sustained RAM usage over a month. Basic and Power User plans come with 600 compute units, which is more than most users need.
